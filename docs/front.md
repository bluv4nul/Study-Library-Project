# ElectroLibrary Frontend

## Описание

Frontend-часть проекта **ElectroLibrary**, реализованная с использованием **Vue 3**, **Vue Router** и **Axios**.

Приложение предоставляет удобный пользовательский интерфейс для взаимодействия с серверной частью электронной библиотеки и позволяет выполнять основные операции управления коллекцией книг.

Дизайн интерфейса выполнен в минималистичном стиле.

## Функциональные возможности

### Работа с книгами

* Просмотр списка книг.
* Просмотр подробной информации о книге.
* Создание новых записей.
* Редактирование существующих книг.
* Удаление книг из коллекции.

### Управление справочниками

* **Авторы**: добавление, редактирование и удаление авторов библиотеки.
* **Жанры**: управление жанрами для категоризации книг.
* **Издательства**: управление информацией об издательствах.

### Поиск и фильтрация

* Поиск книг по названию.
* Фильтрация по автору.
* Фильтрация по издательству.
* Фильтрация по жанру.
* Сортировка списка книг по различным параметрам.
* Поиск и сортировка авторов, жанров и издательств.

### Навигация

Для организации многостраничного интерфейса используется Vue Router.

Доступные маршруты:

| Маршрут        | Описание                                |
| -------------- | --------------------------------------- |
| `/`            | Перенаправление на список книг          |
| `/books`       | Список всех книг                        |
| `/books/:id`   | Подробная информация о книге            |
| `/authors`     | Управление авторами                     |
| `/genres`      | Управление жанрами                      |
| `/publishers`  | Управление издательствами               |

## Компонентная архитектура

### BookCard

Компонент отображения информации о книге в виде карточки.

Функции:

* отображение названия книги;
* отображение автора;
* отображение года издания;
* переход к подробной информации.
EntitySelect

Компонент с умным выбором и автодополнением для выбора авторов, жанров и издательств.

**Props:**
- `items`: массив доступных элементов
- `modelValue`: выбранное значение (строка или массив строк)
- `isMultiple`: поддержка множественного выбора (boolean)
- `labelField`: название поля для отображения
- `placeholder`: подсказка в поле ввода

**Events:**
- `@update:modelValue`: обновление выбранного значения
- `@createNew`: запрос на создание нового элемента

**Возможности:**
- Поиск по введённому тексту с регионезависимой фильтрацией
- Подсказки при вводе (автодополнение)
- Создание новых элементов если они не найдены (кнопка "новый")
- Визуальное отображение выбранных элементов в виде тегов для множественного выбора
- Поддержка множественного и одиночного выбора

### BookForm

Компонент формы создания и редактирования книги.

**Обязательные поля:**
* Название книги
* Авторы (один или несколько)
* Жанры (один или несколько)

**Опциональные поля:**
* Год издания
* ISBN
* Издательство

**Особенности:**
* Поля для авторов, жанров и издательства используют `EntitySelect` компонент
* Возможность быстрого создания новых авторов, жанров и издательств прямо из формы
* Поддержка множественного выбора авторов и жанров
* При создании нового элемента через автодополнение, он сразу становится доступным для выбора

**Events:**
- `@submit`: отправка формы
- `@cancel`: отмена редактирования
- `@createAuthor`: запрос на создание нового автора
- `@createGenre`: запрос на создание нового жанра
- `@createPublisher`: запрос на создание нового издательства

### BaseModal

Универсальное модальное окно.

**Props:**
- `modelValue`: видимость окна (v-model)
- `title`: заголовок модального окна
- `subtitle`: подзаголовок

Использует механизм слотов Vue для передачи содержимого.

### PageHeader

Переиспользуемый компонент заголовка страницы.

**Props:**
- `eyebrow`: категория/описание страницы
- `title`: основной заголовок
- `description`: описание содержимого

**Slots:**
- `actions`: область для кнопок действий

## Навигация и интеграция

### Главное меню (App.vue)

В шапке приложения находятся основные ссылки навигации:
- **Книги** → `/books`
- **Авторы** → `/authors`
- **Жанры** → `/genres`
- **Издательства** → `/publishers`

Отсюда пользователь может быстро перейти к любому разделу приложения.

### Рабочие процессы

#### Создание книги с новыми элементами справочников

1. На странице /books нажать **"+ Новая книга"**
2. В форме добавления:
   - Ввести название (обязательно)
   - В поле "Авторы" начать вводить имя автора
   - Если автор не найден → нажать **"+ Новый"** для создания
   - Новый автор сразу появится в списке и будет выбран
   - Повторить для жанров и издательства
3. Нажать **"Создать книгу"**

## Использование слотов

В проекте используются слоты Vue для повышения гибкости компонентов.

Пример:

```vue
<BaseModal v-model="isOpen" title="Заголовок">
  <template #default>
    <BookForm />
  </template>
</BaseModal>
```

Использование слотов позволяет переиспользовать модальные окна и другие компоненты для различных задач.
